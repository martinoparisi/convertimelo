ISTRUZIONI PER AVVIARE IL SERVER PYTHON LOCALE

Poiché non puoi usare le Cloud Functions (richiedono piano a pagamento), abbiamo creato un server Python locale che puoi eseguire sul tuo computer.

1. INSTALLA LE DIPENDENZE
Apri un terminale e installa le librerie necessarie:
pip install flask flask-cors pint google-generativeai

2. IMPOSTA LA CHIAVE API (Per Genkit/AI)
Se usi Windows (PowerShell):
$env:GOOGLE_API_KEY="la-tua-chiave-api-qui"
Oppure CMD:
set GOOGLE_API_KEY=la-tua-chiave-api-qui

3. AVVIA IL SERVER
Esegui questo comando dalla cartella principale del progetto:
python convertimelo/functions/local_server.py

Il server partirà all'indirizzo http://0.0.0.0:5000 (accessibile da tutta la rete locale).

4. USA L'APP
L'applicazione Angular è già stata configurata per puntare al tuo IP locale (192.168.1.20:5000).
Ora puoi usare i convertitori anche dal telefono, purché il server Python rimanga aperto sul PC.
